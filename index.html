Git Fetch:

Git Fetch command remote repository se latest changes ko local repository mein laata hai, lekin ye changes aapke current branch mein merge nahi karta. Iska matlab hai ki aapko manually merge karna padega agar aap chahte hain ki ye changes aapke current branch mein reflect ho.
